# Maying
A ShadowsocksRR client for Android, written in Java.
Modified from [https://github.com/xhvay/ShadowsocksRR](https://github.com/xhvay/ShadowsocksRR)



### PREREQUISITES

* JDK 1.8
* AndroidStudio 3.5
* Android Build Tools 29+
* Android NDK 19+
